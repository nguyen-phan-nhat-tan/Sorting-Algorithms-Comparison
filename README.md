# Sorting-Algorithms-Comparison
Báo cáo này trình bày kết quả thử nghiệm và đánh giá hiệu năng của các thuật toán sắp xếp phổ biến bao gồm Quicksort, Heapsort, và Mergesort, so sánh trực tiếp với hàm numpy.sort() của Python. Thử nghiệm được thực hiện trên bộ dữ liệu quy mô lớn gồm 10 dãy số, mỗi dãy chứa 1.000.000 phần tử.
